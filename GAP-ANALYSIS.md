### 2. 技术层面

#### 2.1 后端技术选型

| 技术点 | 当前状态 | 建议 |
|--------|---------|------|
| Node.js 版本 | ❌ | 建议 v18+ LTS |
| Express 版本 | ❌ | 建议 v4.x |
| 数据库 | ⚠️ | 设计用 SQLite，建议考虑 PostgreSQL |
| ORM | ❌ | 建议使用 Prisma / Sequelize |
| 数据验证 | ❌ | 建议使用 Joi / Zod |

| 配置管理 | ❌ | 建议使用 dotenv + config 库 |

**建议技术栈**:

  "winston": "^3.0",

  "dotenv": "^16.0",
  "cors": "^2.8",
  "helmet": "^7.0",
